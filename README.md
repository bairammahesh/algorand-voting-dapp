# Algorand Fundamentals Project

## Overview
This repository is created for the Algorand Fundamentals course assignment.

The goal of this repository is to build and maintain a project using the Algorand blockchain.

## Project Idea
Simple decentralized voting application built on Algorand.

## Features
- Secure blockchain voting
- Transparent vote counting
- Decentralized storage

## Technologies
- Algorand Blockchain
- Python
- Algorand SDK
